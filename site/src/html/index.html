Problem

Libraries exist that encapsulate each of these tasks, and conquer them individually. However, loading multiple libraries that all operate on scroll typically introduces more problems than it solves:

  • Code bloat
  • Performance bottlenecks
  • Mandatory HTML/CSS additions

Solution

Bricks provides the minimal JavaScript required to create scroll based interactions. It features:

  • Code bloat
  • Performance bottlenecks
  • Mandatory HTML/CSS additions
